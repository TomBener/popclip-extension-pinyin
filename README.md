# My PopClip Extensions

## Usage

```
Usage:
	./build.sh [-c | -i] <EXT_NAME>
	./build.sh -h

Options:
	-c	Create an extension by template
	-i	Build and install
	-h	Show help
```

## Extensions Index

- Open GitHub Repo

  > Open GitHub Repo in a web browser with the selected text like `AUTHOR/REPO`
